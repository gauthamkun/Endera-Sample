Instructions:

Please upload your annotated files into this folder.

To better organize the data please name the data by the naming convention of: [dataset name]_[records analyzed]_[tags used].

Example: Auburn_1-2000_Drugs-Sex-Assualt

Remember that after uploading the files into the repository you will need to commit the change then push it.

Please name the commit the same as your file.

Be sure to pull any new changes first before pushing.

Please continue to document your time, as well as the number of records analyzed on the confluence site, but keep all files contained to the repository.
